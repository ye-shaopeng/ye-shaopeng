# 文献：linux的vi和vim编辑器

- 三种模式：
    - 一般模式[^1]：默认，通过`esc`返回到一般模式。
        - `Ngg`：将光标，移动到第`N`行。
            - `gg`：等价于`1gg`。
        - `^`：将光标，移动到行首。
        - `$`：将光标，移动到行尾。
        - `Nyy`：从光标所在行（包含）开始，复制`N`行。
            - `yy`：等价于`1yy`。
        - `Nyw`：从光标所在字符（包含）开始，复制`N`个单词。
            - `yw`：等价于`1yw`。
        - `Np`：1）往光标所在行后；2）往光标所在字符后，粘贴`N`次。
            - `p`：等价于`1p`。
        - `Ndd`：从光标所在行（包含）开始，删除`N`行。
            - `dd`：等价于`1dd`。
        - `Ndw`：从光标所在字符（包含）开始，删除`N`个单词。
            - `dw`：等价于`1dw`。
        - `Nu`：撤销`N`次操作。
            - `u`：等价于`1u`。
        - `N + CTRL + r`：恢复`N`次操作。
            - `CTRL + r`：等价于`1 + CTRL + r`。
    - 编辑模式
        - `O`：往光标所在行，前插一空行，开始编辑。
        - `o`：往光标所在行，后插一空行，开始编辑。
        - `I`：将光标移动到行首，开始编辑。
        - `i`：从光标所在字符前，开始编辑。
        - `A`：将光标移动到行尾，开始编辑。
        - `a`：从光标所在字符后，开始编辑。
    - 指令模式
        - `:wq!`：强制保存并退出。
            - `!`：强制。
            - `w`：保存。
            - `q`：退出。
        - `/WORD`：查找`WORD`。
            - `N`：前一个。
            - `n`：后一个。
            - `:noh`：去除高亮。
        - `:set nu`：显示行号。
        - `:set nonu`：不显示行号。
        - `:%s/OLD/NEW/g`：查找并替换。 #todo
