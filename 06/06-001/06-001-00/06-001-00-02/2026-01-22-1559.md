# 文献：linux的系统管理操作-关闭防火墙

- 系统管理操作——查看服务的状态、启动服务、停止服务、服务开机自启。
- 管理系统服务： #tocheck
    - `ls -l /usr/lib/systemd/system`
    - `systemctl list-unit-files`
    - `systemctl status SERVICE`
    - `systemctl stop SERVICE`
    - `systemctl is-enabled SERVICE`
    - `systemctl disable SERVICE`
- 停止firewalld服务并禁止开机自启。 #tocheck
- 关闭windows的防火墙（推荐）：
    - [ ] 控制面板 > 系统和安全 > Windows Defender 防火墙 > 启用或关闭Windows Defender 防火墙
        - [ ] 专用网络设置
            - [ ] 关闭Windows Defender 防火墙：`√`
        - [ ] 公用网络设置
            - [ ] 关闭Windows Defender 防火墙：`√`
