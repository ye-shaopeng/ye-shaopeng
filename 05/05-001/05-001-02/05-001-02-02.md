# 传入类型实参

1. **只要使用[^3][^4]泛型类，就要传入类型实参。**
2. *声明什么泛型 => 传入什么类型实参*：
    - 声明`泛型 extends 上界`，传入`上界`或`上界子类`，可以调用`上界方法`。 ^db92fb
3. 根据[*具体类型*](05-001-02-02-02.java.md)：
    1. “写”^[泛型用来表示，方法的参数类型。]：编译时，**检查实参类型（具体类型、具体类型子类）**。 ^37f4a9
    2. “读”[^5]：编译时，**强制类型转换（具体类型[^6]）**。
