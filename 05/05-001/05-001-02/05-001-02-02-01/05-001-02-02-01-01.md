# 通配符`?`

#tolink

| 编译时类型       | 运行时类型             | 上边界 => “读” | 下边界 => “写”         |
| ---------------- | ---------------------- | -------------- | ---------------------- |
| `?`              | `Object`、`Object子类` | `Object`       | `null`                 |
| `? extends 上界` | `上界`、`上界子类`     | *`上界`*       | `null`                 |
| `? super 下界`   | `下界`、`下界父类`     | `Object`       | *`下界`*、*`下界子类`* |
