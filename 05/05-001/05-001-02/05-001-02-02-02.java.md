# **具体类型 -> 泛型 -> 其他泛型……**

```java
public interface List<T extends Object> {

    void add(int index, T element);

    T remove(int index);

}
```

```java
// 泛型 -> 其他泛型
public class DynamicArray<E extends Object> implements List<E> {

    @Override
    public void add(int index, E element) {
        ...
    }

    @Override
    public E remove(int index) {
        ...
    }

}
```

```java
// 具体类型 -> 泛型 -> 其他泛型
public class DynamicArrayTest {

    public static void main(String args[]) {
        DynamicArray<Integer> list = new DynamicArray<>();

        // 编译时，检查实参类型（具体类型、具体类型子类）
        // list.add(int index, Integer element)
        list.add(0, 0);

        // 编译时，强制类型转换（具体类型）
        Integer i = list.remove(0);
    }

}
```

对比：

```java
// 具体类型 -> 泛型
public class IntegerArray implements List<Integer> {

    @Override
    public void add(int index, Integer element) {
        ...
    }

    @Override
    public Integer remove(int index) {
        ...
    }

}
```
