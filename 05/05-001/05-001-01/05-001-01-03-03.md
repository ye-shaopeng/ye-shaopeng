# `!(A && B) == (!A || !B)`

- `!(A && B)`：考虑，除了`A && B`，所有情况： #todo
    - ① `!A && B`
    - ② `!A && !B`
    - ③ `A && !B`
- `!(!A || !B) == !(!(A && B)) == (A && B)`
- 具体使用：
    - 已经处于`A && B`的状态：`if (A && B) { ... } else { if (!A) { ... } else { ... } }`
    - 还未处于`A && B`的状态：`while (A && B) { ... } if (!A) { ... } else { ... }`
